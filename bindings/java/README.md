# iota.rs-java

This is the **alpha** version of the official JAVA binding to [IOTA's Rust API](https://github.com/iotaledger/iota.rs).
Inspired by https://github.com/drrb/java-rust-example.

## Building

```bash
$ mvn package
```

## Running tests
```bash
$ mvn test
```

## Running example

```bash
$ java -jar target/iota-client.jar
```
